🛠️ איך יצרנו את האתר?

שלב 1: פתיחת המאגר ב-GitHub

כדי שהאתר שלנו יהיה באוויר ובחינם, השתמשנו בשירות שנקרא GitHub Pages. ככה עושים את זה:

  1. נרשמים ופותחים משתמש חינמי באתר GitHub.
  2. לוחצים על הכפתור הירוק New Repository כדי ליצור "תיקיית עבודה" חדשה ברשת.
  3. הסוד הוא בשם המאגר: כדי שהאתר יקבל את הכתובת הראשית, חובה לקרוא למאגר בדיוק בשם המשתמש שלכם פלוס הסיומת, למשל: username.github.io.
  4. מוודאים שהמאגר מוגדר כ-Public (פומבי) ולוחצים Create.

מה זה קובץ README?

קובץ ה-README.md הוא בעצם "עמוד השער" או תעודת הזהות של הפרויקט שלכם ב-GitHub.

כשמישהו נכנס למאגר הקוד שלכם, זה הדבר הראשון שהוא קורא. הוא נועד להסביר במילים פשוטות למה האתר הזה נועד, מי בנה אותו, ואיך להשתמש בו.

הקובץ נכתב בשפה קלה שנקראת Markdown, המאפשרת לעשות כותרות והדגשות בעזרת סימנים פשוטים.

שלב 3: כתיבת הקוד (index.html)

השלד של כל אתר אינטרנט נקרא HTML. יצרנו קובץ בשם index.html.

כדי ליצור כפתור שמקשר למקום אחר, השתמשנו בתגית הקישור. הנה דוגמה לאיך שכפתור יוטיוב בנוי בקוד (השתמשנו בסימנים מיוחדים כדי להציג את זה):

<a href="https://youtube.com" class="link-btn neon-red">
    YouTube - הערוץ הרשמי 📺
</a>

בתוך ה-href שמים את הקישור, ובתוך ה-class שמים את המחלקות שאחראיות על העיצוב.

שלב 4: עיצוב ואפקטים של ניאון

כדי שהאתר לא ייראה משעמם, השתמשנו ב-CSS (בחלק העליון של הקוד).

כדי ליצור את אפקט הזוהר כשמעבירים את העכבר, השתמשנו ב-box-shadow בשילוב עם :hover. הנה הקוד שעושה את הקסם:

.neon-red:hover {
    border-color: #ff0050;
    box-shadow: 0 0 15px #ff0050, 0 0 30px #ff0050;
    color: #ff0050;
}
